1. 변수(Variable)
변수 : 변하는 값, 데이터를 담는 공간(메모리)

변수 선언
    var a;
    - 변수 선언 후 값을 할당하지 않으면 기본적으로 초기값은 undefined

    var b;
    b = 10;

변수의 타입 확인 : typeof
