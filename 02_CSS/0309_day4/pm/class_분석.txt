1. 글꼴의 크기를 지정하는 단위
    rem : 상대단위 = 설정하면 해상도에 따라 크기가 유동적으로 변함
        - html 요소의 기본값이 16px이므로 기본 상태에서 1rem = 16px
        - html 요소의 값이 바뀌면 상대적으로 조절됨

    px : 절대 단위 = 페이지 크기를 줄이든 키우든 같은 크기

2. overflow 속성 : 내용이 요소의 크기를 벗어났을 때 어떻게 처리할지
    - CSS는 기본적으로 visible (상자 크기와 상관 없이 요소를 보여줌)
    - hidden 적용 시 넘치는 콘텐츠는 보이지 않음
