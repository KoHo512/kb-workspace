레이아웃 : 이미지, 텍스트 등의 요소들을 원하는 위치로 배치하는 것

1. 플렉스 박스(Flex Box)
    - UI 디자인에 최적화된 레이아웃을 정의하는 CSS

2. 플렉스 박스가 필요한 순간은?
3. 플렉스 박스로 정렬
4. 플렉스 박스로 축 방향 바꾸기


플렉스 박스 제 1원칙 : 부모만 플렉스할 수 있다
    - 부모자식 관계를 파악한 후 부모에 적용해야 자식이 영향을 받음


플렉스 박스 사용 방법
    1. 공간에 맞추기 : display flex
        - display 속성 5가지 : block, inline, flow, grid, flex
        - flex는 자식 요소를 원하는 방향으로 유연하게 배치할 수 있는 속성

    2. 주축 정렬 : justify-content 속성
        - 웹페이지 축을 따라 요소 사이에 공간 만드는 방법 정의
        - 축 : 주축(가로), 교차축(세로)
        - flex-direction 속성 사용해 축 변경 가능

        - 플렉스 박스는 두 축(주축, 교차축)을 기준으로 움직이기 때문에 축 방향을 기준으로 요소를 움직임

    3. 교차축 정렬 : align-items 속성
    
    4. 플렉스 축 방향 바꾸기 : flex-direction 속성
        - column(세로 방향), row(가로 방향 - 기본값), row-reverse(역순으로 배치)
