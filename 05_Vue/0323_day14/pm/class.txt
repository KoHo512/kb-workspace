Vue3 핵심 문법
- Vue 컴포넌트를 하나의 파일로 나타내어 사용됨 (SFC: Single File Component)
- 하나의 파일이 하나의 컴포넌트를 나타내므로 관리 용이, 코드 간결


SFC 구성 요소
- Template (<template>...</template>) : 컴포넌트가 렌더링되는 HTML 코드 부분
    - 선언적 렌더링 또는 템플릿 문법을 이용해 반응형 컴포넌트를 만들 수 있음

- Script (<script>...</script>) : Template의 반응형 컴포넌트를 제어하는 영역 (변수 조작 등)
    - 타입스크립트나 자바스크립트를 통해 스크립트 코드 작성

- Style (<style>...</style>) : 컴포넌트 또는 전체 프로젝트에서 사용할 CSS 작성
    - <style scoped> 제한 가능



[실습] 라이프사이클 훅
** 요구사항
- 컴포넌트가 생성되면 사용자 이름 가져오기
- DOM이 렌더링된 후 "화면 준비 완료" 로그 출력하기
- 버튼 클릭 시 데이터 다시 가져오기
=> 어떤 훅을 써야 할까? created, mounted

** 요구사항
- 박스(div)의 너비를 화면에 출력
- 컴포넌트 생성 시(created)에 시도
- 렌더링 후(mounted)에 시도
- 두 결과 비교
