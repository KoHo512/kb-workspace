calculator 계산기 만들기

1. UI (사용자 화면)
2. 데이터 바인딩하고 이벤트 연결
    - 기능 구현
        - 계산기 UI에서 숫자(Operand), 사칙연산(Operator) 입력
        - 입력된 데이터를 출력
        => 입력값을 저장하는 데이터 정의
            필요한 부분을 바인딩 처리

    1. App.vue
        1-1. 데이터(변수) 정의
            - 출력데이터 저장 : output
            - 이전에 입력된 숫자나 연산 결과 저장 : prev
            - 현재 입력 중인 숫자 저장 : cur
            - 사용자가 입력한 연산자 저장 : operator

        1-2. 데이터 중 HTML 요소와 바인딩되어야 할 데이터는?
            - 사용자에게 계속 노출되는 데이터 = output => 양방향 바인딩 처리

        1-3. 이벤트 연결
            - 계산기 UI에 필요한 이벤트 연결
            - output 영역을 제외한 숫자 10개, ., (+, -, *, /), C 버튼이 이벤트 연결 대상

            - 클릭 이벤트 처리(@click)를 연결
            - 클릭 이벤트에 따른 이벤트 핸들러 작성 : operation()


    2. 계산기 로직 구현
        - logic : 프로그램이 수행해야 하는 작업 또는 계산 순서와 조건을 정의하는 과정

        - 계산을 담당하는 로직은 클릭 이벤트에 연결한 operation() 내부에 작업

        2-1. 클릭한 버튼 값 가져오기
            - <input>의 value 속성에 할당된 값을 가져오기
            - input 속성의 값을 이벤트 객체가 가져와서 출력하는 코드 작성

        2-2. 입력값 e.currentTarget.value를 output에 추가
            - 이벤트가 발생한 UI 값을 출력칸에 노출되도록 로직 추가

            1. 버튼을 클릭하면 해당 숫자를 n 변수에 저장
                - 이때 저장된 숫자는 내부적으로는 문자열 데이터

            2. 사용자가 입력한 숫자를 cur 데이터에 저장
                - 자바스크립트에서는 문자열끼리 덧셈하면 문자열이 하나로 연결
                - cur 데이터를 output에 저장

            3. 연산자(+, -, *, /) 분리 작업

            4. = 눌렀을 때 계산
                1. cur 안에 있는 연산자 찾기
                2. 연산자를 기준으로 왼쪽 숫자와 오른쪽 숫자 나누기
                3. 문자열 숫자를 실제 숫자로 변환 : parseInt()
                4. 연산식 수행
                5. 결과를 output과 cur에 저장

            5. eval(), 직접 파싱 후 출력
