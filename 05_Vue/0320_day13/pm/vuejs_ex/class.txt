    - 컴포넌트 : 어플리케이션을 구성하는 재사용 가능한 요소(단위)

    [실습 1]
        - 고객 요청 1 : 앞바퀴 휠과 타이어를 새로 장착해 주세요 (타이어 파랑)
        - 고객 요청 2 : 뒷바퀴 휠과 타이어를 새로 장착해 주세요

        자동차 작업 순서
        - 앞바퀴 휠 장착 => 뒷바퀴 휠 장착 => 타이어 장착 => 타이어 커스터마이징 (파랑)
    
    App.vue : 모든 컴포넌트의 부모, 최상위 컴포넌트 = Root

    [문제] Root가 언제 임포트되어 사용되었을까?
    - 답 : Vue 어플리케이션에서 처음 실행되는 src/main.js가 실행되었을 때

    Vue의 실행 순서
    1. 웹 브라우저에서 index.html 로드
    2. 로드하는 중, src/main.js 파일 로드하고 실행
    3. 실행 중, Root인 App.vue를 임포트하고 사용

1. 기본 템플릿 문법
    1-1. 텍스트 출력
        머스태시 문법 : Vue에서 텍스트를 동적으로 출력하는 대표 방법 - {{ }}
        v-text 디렉티브를 이용하여 동적 출력

        * 디렉티브(Directive) : 템플릿 블록에서 HTML 요소에 입력하는, "v-"로 시작하는 특수 속성
            동적으로 텍스트 출력하거나 클래스 표시, 조건에 따라 일부 요소 출력 제어 반복, 이벤트를 다루는 역할 등등

        * JSON(JavaScript Object Notation) : 가볍고 직관적인 문자열 형태로 만들어진 데이터 형식
            데이터 저장과 교환에 사용

            JavaScript 애플리케이션에서 Java 애플리케이션으로 배열 데이터를 전달할 때
            두 언어의 배열 처리 방식이 달라 직접적으로 호환되지 않음
            배열 데이터를 JSON으로 변환하면 JavaScript와 Java에서 함께 사용 가능

    - 반응형 상태
    - 컴퓨티드
    - 데이터 바인딩
