프로그래밍 언어의 함수
    - 문제를 해결하는 과정을 문(statement)으로 구현한 다음, 해당 코드를 블럭({})으로 감싸 하나의 실행 단위로 정의한 것

    - 함수
        - 매개변수(parameter) : 함수 내부로 입력값을 전달받는 변수
        - 입력값(argument)
        - 출력값 = 반환값 (return value)

    - 함수 구분 기준(식별자) : 함수의 이름

---

변수 : 선언(declaration) => 컴파일러한테 전달
    - 변수 이름 : argument
    - 타입 : int
    - 컴파일러한테 변수가 존재하고 메모리 공간이 필요하다고 전달 (실제 메모리 생성되지 않음)

함수 : 정의(definition)
    - 실제 동작(코드)이 있어야 함
        - 함수 이름
        - 매개변수
        - 반환 타입
        - 실행코드
    - 컴파일러한테 이 함수가 이렇게 동작한다고 고지 (= 구현 제공)
