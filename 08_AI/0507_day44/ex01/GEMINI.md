# Pastel Todo App

순수 자바스크립트와 CSS를 사용하여 만든 파스텔 톤의 할 일 목록(Todo List) 애플리케이션입니다.

## 프로젝트 개요
이 프로젝트는 사용자가 일상적인 할 일을 관리할 수 있는 가볍고 직관적인 웹 애플리케이션입니다. 별도의 백엔드 서버 없이 브라우저의 로컬 스토리지를 활용하여 데이터를 저장합니다.

- **핵심 기술**: HTML5, CSS3, Vanilla JavaScript
- **주요 기능**:
  - 할 일 추가 및 삭제
  - 할 일 완료 상태 토글 (체크박스)
  - 할 일 텍스트 수정 (더블 클릭)
  - 필터링 (전체, 진행 중, 완료)
  - 로컬 스토리지 기반 데이터 유지

## 빌드 및 실행 방법
이 프로젝트는 빌드 과정이 필요 없는 정적 웹 애플리케이션입니다.

- **실행**: 웹 브라우저에서 `index.html` 파일을 엽니다.
- **테스트**: 별도의 테스트 프레임워크는 설정되어 있지 않으며, 브라우저 개발자 도구(F12)를 통해 기능을 검증할 수 있습니다.

## 개발 컨벤션
- **언어**: 최신 ECMAScript 표준(ES6+)을 따르는 순수 자바스크립트를 사용합니다.
- **스타일링**: CSS Variables를 사용하여 테마 색상을 관리하며, Flexbox를 주 레이아웃 도구로 사용합니다.
- **상태 관리**: `todos`라는 글로벌 배열을 통해 상태를 관리하며, 상태가 변경될 때마다 `renderTodos()` 함수를 호출하여 UI를 업데이트하는 선언적 패턴에 가까운 방식을 지향합니다.
- **영속성**: 데이터 변경 시마다 `localStorage`에 JSON 형태로 동기화합니다.

## 주요 파일 구조
- `index.html`: 애플리케이션의 구조 정의
- `style.css`: 파스텔 톤 테마 및 레이아웃 스타일 정의
- `script.js`: 할 일 추가, 삭제, 수정, 필터링 등 핵심 비즈니스 로직 구현
