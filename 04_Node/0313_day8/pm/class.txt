비동기 처리 방식
- callback : 요청 종료 후 실행할 함수를 매개변수로 전달하는 방식

- promise : Promise 객체 반환하는 방식
    - 자바스크립트에서 비동기 실행을 동기화하는 구문으로 사용
    - 의미 : 현실에서 약속 = 미래에 어떤 일을 할 거라고 정하는 것
        - 약속은 이행, 거절, 대기 3가지 상태를 갖음

- async/await : Promise를 간단하게 사용하도록 변경한 방식

async => 비동기
await => 기다린다
- 자바스크립트에서 await는 무엇을 기다릴까?
    - 성공/실패로 Promise의 실행이 완료되기를 기다림
    - await 뒤에는 Promise가 옴

    - await는 async 함수 안에서만 사용됨
