1. Node.js : 서버에서 자바스크립트를 사용할 수 있게 제공하는 런타임 환경
    - 브라우저 밖에서 JS를 사용하는 V8 엔진 사용
    - 논블로킹/비동기 API 제공

    - 2010년 npm이라는 패키지 매니저 공개

    - V8 엔진 : C++로 만든 오픈소스 자바스크립트 엔진
        - 엔진 : 사용자가 작성한 코드를 실행하는 프로그램
    - 구성 : 파서, 컴파일러, 인터프리터, 가비지 컬렉터, 콜스택, 힙
        - 자바스크립트 코드 =(파서)=> 추상 구문 트리
        - 추상 구문 트리 =(이그니션)=> 바이트 코드
        - 이그니션 최적화 =(터보팬)=> 바이너리 코드 =(최적화 해제) => 바이트 코드
    
        - JIT(Just-in Time) 컴파일러 = 이그니션(인터프리터) + 터보팬(컴파일러)
            - 속도 빠르고 최적화 가능 / 메모리 많이 사용

    - libuv 라이브러리를 통해 자바스크립트에서 제공하지 않는 기능(HTTP, 파일, 소켓 통신, I/O 등) 제공
