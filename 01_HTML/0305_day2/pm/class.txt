웹페이지는 트리 구조로 구성되어 있다

검색엔진 : 웹에서 원하는 정보를 찾는 데 걸리는 시간을 최소화하기 위해 사용되는 시스템(프로그램)

봇(bot), 크롤러 : 사용자가 검색했을 때 결과를 빠르게 보여주기 위해 정보를 미리 데이터베이스화 시키는 프로그램

검색엔진 최적화(SEO) : 검색이 잘 되도록 설계하는 일

HTML로 웹페이지를 만들면서 검색 엔진 최적화하는 방법

1. 메타 태그 : 검색 엔진에 정보를 제공할 목적으로 사용하는 태그
2. 시맨틱 태그 : 웹페이지 구조를 구성하는 태그

레이아웃용 시맨틱 태그
- <header> : 소개 요소 또는 탐색 링크
- <nav> : 탐색 링크 정의 - 주로 메뉴
- <section> : 본문 내용 포함하는 공간
- <article> : 독립적인 요소
- <aside> : 좌우측에 위치하는 공간 (사이드바)
- <footer> : 문서 하단을 정의 - 카테고리 요약, 저작권 표시 등
